# PLANTILLA — Nuevo proyecto/cliente

> Copia esta carpeta a proyectos/<nombre-cliente>/ cuando entre un cliente nuevo.
> Cada proyecto necesita estos archivos:

- ESTADO.md       → dónde vamos hoy (lo primero que se pega en cada sesión)
- DECISIONES.md   → qué decidimos y por qué
- DUDAS.md        → lo que no entiendo
- BITACORA.md     → qué pasó cada día
- MISE_EN_PLACE.md→ apps, APIs, configs (SIN claves)
- CREDENCIALES.md → índice que apunta a Bitwarden (NUNCA la clave real)

## Prompt para arrancar un cliente nuevo con Fable:
"Cliente nuevo: [nombre]. Es [tipo de negocio]. Te paso [web/rrss/contexto].
Crea la carpeta de proyecto desde la plantilla y dime qué le podemos ofrecer."
